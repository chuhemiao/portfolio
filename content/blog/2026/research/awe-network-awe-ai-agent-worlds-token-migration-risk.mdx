## Pre-screen Decision

Full research. AWE has major exchange coverage, a new AI-agent infrastructure narrative, and a token migration history from STPT. It is exactly the kind of mid-cap narrative token where source discipline matters.

## TL;DR

AWE Network, formerly STP Network, is building the Autonomous Worlds Engine for on-chain AI multi-agent worlds and gaming/developer infrastructure ([AWE Network](https://www.awenetwork.ai/)). As of June 28, 2026, Surf shows AWE at about $0.063, $122.6M market cap/FDV, and about $8.0M 24h volume. The token is broadly listed on Binance, Upbit, Coinbase, Bithumb, and Bitget, which gives it strong market access.

Bottom line: AWE is an AI/gaming infra watchlist token. The bull case requires real developer adoption of agent-world tooling. The bear case is that the STPT-to-AWE migration creates a fresh narrative without enough usage-based value capture.

## Project Overview

Surf describes AWE as a Base-based project focused on developer tooling, gaming infrastructure, and Web3 AI. The project GitHub still points to the prior STP developer organization ([STPDevteam](https://github.com/STPDevteam/)), which is a useful reminder that this is not a brand-new launch but a migrated/repositioned asset.

## Research Question

Is AWE becoming a real AI-agent world infrastructure layer, or is it primarily a rebrand that monetizes the AI narrative through exchange liquidity?

## Architecture and Mechanism

The product thesis is a modular engine for autonomous worlds: agent frameworks, game-like environments, and tooling for developers to build multi-agent on-chain experiences. The token thesis depends on ecosystem usage, transaction/payment flows, and whether AWE is required for access, incentives, or governance. Without direct usage demand, AWE behaves like a liquid AI narrative proxy.

## Economics and Value Capture

AWE value capture is not proven by the token migration. It needs one or more durable flows: developers paying to deploy agent worlds, users spending inside autonomous worlds, token-gated infrastructure access, or staking/governance with real economic rights. If the token only represents ecosystem vibes, value capture is weak.

## Team and Funding

Surf lists Mike Chen and Richard Lee as co-founders and reports about $9.75M raised across IEO/private rounds, including Neo, Alphabit, and Block Capital. That history helps with execution credibility, but also means investors should demand evidence that the new AWE strategy is more than a repackaging of old STP distribution.

## Red-team Check

The strongest bear case is that AWE is a successful ticker migration, not a successful infrastructure network. The most gameable metric is AI-agent narrative engagement. The value-capture failure path is developer tools with no required token demand. The zero path is failed migration trust, no apps, and exchange liquidity drying up after the AI rotation.
