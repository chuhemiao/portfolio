### 2. 整体难度

整体难度属于 F-G 级稳定基础盘：单点知识不深，但跨题型复现密度很高。命题不追求复杂语法，而追求“基础表达是否真的会用”。

难点集中在四处：

| 难点           | 命题表现                                                   | 学生常错                   |
| -------------- | ---------------------------------------------------------- | -------------------------- |
| 助词语义       | 同一动词搭配不同格，如 `で書く`、`を出る`、`に貼る`        | 只背中文意思，不看动词支配 |
| 疑问词与指示词 | `だれ / どなた / どこ / どちら / いくら / いくつ` 反复出现 | 被中文“哪个/哪里/多少”混淆 |
| 听力最终结论   | 前面信息被否定或改口，答案在最后行动                       | 只抓关键词，没听最终决定   |
| 阅读选项改写   | 原文简单，选项换说法或设“与内容不符”                       | 只定位词，不核对逻辑       |

### 3. 与历年常见考法的关系

这 32 回显示出 F-G 级强烈的“题型模板化”特征：

| 命题偏好                 | 说明                                                     |
| ------------------------ | -------------------------------------------------------- |
| 固定题位复现             | 每回题型、题数、分值几乎不变                             |
| 高频基础点轮换           | 助词、疑问词、时间数量、动词接续每回都有                 |
| 表面换场景，底层不换     | 购物、学校、医院、车站、家庭、天气、饮食反复换壳         |
| 阅读和听力共享能力       | 时间、数量、地点、人物行动、最终决定同时出现在读解和听解 |
| 干扰项多为“语义类型错位” | 问地点给人物，问金额给时间，问原因给动作                 |

### 7. 短期最值得投入的提分点

优先顺序：

1. 听力应答 448 题槽：投入产出最高，规则明确。

4. 会话最终行动和数字题：短期通过记笔记模板提分明显。
5. 句序题：掌握骨架法后正确率提升快。
6. 汉字读写：每天小量高频复现，稳分。

## 十一、四个成果总结

### 1. J-TEST 历年高频考点地图

核心地图：疑问词/こそあど、助词、动词接续、数字时间、阅读核对、听力应答、最终行动、句序、基础汉字。

### 2. J-TEST 相同考点对照表

相同考点主要不是“同一道题重复”，而是同一底层能力在不同场景中变形：助词从文法题迁移到句序；疑问词从文法迁移到听力应答；数字从汉字、阅读、会话金额题中跨题型出现。

### 3. J-TEST 命题规律与干扰项模型

命题者偏好用日常场景包装基础语法，用相似词、早出现信息、条件切换、最终改口制造干扰。F-G 级真正拉分的不是难语法，而是能否稳定处理“问什么、谁做什么、什么时候、最后怎样”。

### 4. J-TEST 复习优先级与提分路径

最优路径是：听力应答先提分，助词和疑问词打底，动词接续与句序联训，阅读和会话用信息筛选模板，汉字每天稳定复现。短期不应把大量时间投向敬语、使役被动等低频复杂语法。
